# Simple Calculator Web App

A responsive, browser-based calculator built using **HTML, CSS, and JavaScript**.  
It supports keyboard input, light/dark themes, calculation history, and basic input validation.

---

## Features

- Basic arithmetic operations: `+`, `−`, `×`, `÷`, `%`
- Keyboard support (numbers, operators, Enter, Backspace, Escape)
- Light / Dark theme toggle
- Calculation history (last few entries)
- Input validation to prevent invalid expressions
- Responsive design (works on mobile and desktop)
- Smooth hover and button click animations

---

## Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling, responsive layout, themes  
- **JavaScript** – Calculator logic and interactions  

No external libraries or frameworks are used.

## Keyboard Controls

| Key | Action |
|----|--------|
| `0 – 9` | Enter numbers |
| `+ - * / %` | Arithmetic operators |
| `.` | Decimal point |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last character |
| `Esc` | Clear display |
